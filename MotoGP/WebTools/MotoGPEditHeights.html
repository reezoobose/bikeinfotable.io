Description

Use this allows you to compare and modify the height paths of the track edges.

You may need to scroll out in the window after opening the files to make them display.

How To use

  • load the .csv data from the tracer tool [trackname]_FromKml.csv
  • optionally load GPS data to compare against
  • You can drag the node points representing the left and right edges of the track to adjust heights
  • You can left click on the spline to insert a new node
  • You can delete a node by dragging another node into it
  • save the [trackname]_heightsEdited.csv amd [trackname]_heightsEdited.obj
  • next go load these files into MotoGP InterpolatePathss
