How To Use

  • first click out a kml file in MotoGP Track Tracer
  • the use MotoGP Edit Heights to adjust the height profile of the data
  • then use this tool to automatically remove unnecessary extra points in the clicked data
  • then use MotoGP Interpolate Path automatically interpolate extra points in the renderable geometry
  • export the new .obj file, it will be written to your downlaods folder
  • load this file into max and export for unity
