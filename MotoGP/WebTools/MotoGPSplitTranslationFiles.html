Translations are all stored in the "MotoGPMasterStringTable" on Google Docs

To create the latest version of strings to go into the game. Complete all these steps every time.

  1. delete all translationlist*.txt from your downloads folder
  2. open the google doc:(click here to open it),
  3. in google docs select File>>Download as>>Tab-separated values (.tsv, current sheet)
  4. in this tool [Choose File] and select the .tsv file from your downloads folder
  5. this tool will save the following files to your downloads folder
    • translationlist*.txt
  6. include all these files in the Unity project for MotoGP in Assets/Resources
