BikeInfoTable data is stored in the "MotoGP BikeInfo Table" Google Doc

To create the latest version of BikeInfo.cs to go into the game. Complete all these steps every time.

  1. delete all BikeInfoTable.cs from your downloads folder
  2. open the google doc:(click here to open it),
  3. in google docs select File>>Download as>>Tab-separated values (.tsv, current sheet)
  4. in this tool [Choose File] and select the .tsv file from your downloads folder
  5. this tool will save BikeInfoTable.cs to your downloads folder
  6. include this file in the Unity project for MotoGP
